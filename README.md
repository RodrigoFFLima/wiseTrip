A cross-platform mobile app built with .NET MAUI to estimate trip costs based on distance, fuel and tolls.

## 📱 Features

- Calculate trip distance  
- Estimate fuel cost  
- Include toll costs  
- Show total trip cost  

## 🛠️ Tech Stack

- .NET 9  
- .NET MAUI (iOS, Android, MacCatalyst)  
- ASP.NET Core API  

## 🚀 Running the project

### Backend

cd TripWise.Api  
dotnet run  

### Mobile App

cd TripWise.App  
dotnet build
